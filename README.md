# Как получить .bin без установки чего-либо на свой компьютер

Всё, что нужно — бесплатный аккаунт на GitHub и браузер. Компиляция полностью происходит
в облаке (GitHub Actions), скачиваете готовый `.bin`.

## Шаги

1. Зайдите на https://github.com → создайте новый **приватный** (или публичный) репозиторий,
   например `tpager-gps-tracker`.
2. Загрузите в него содержимое этой папки целиком, сохранив структуру:
   ```
   .github/workflows/build.yml
   gps_wifi_tracker/gps_wifi_tracker.ino
   ```
   Проще всего — через веб-интерфейс: **Add file → Upload files**, перетащите обе папки
   (`.github` и `gps_wifi_tracker`) и нажмите **Commit changes**.
3. Как только вы закоммитите файлы в ветку `main`, GitHub автоматически запустит сборку.
   Откройте вкладку **Actions** в репозитории — там будет запущенный workflow
   **"Build T-LoRa Pager firmware"**.
4. Дождитесь зелёной галочки (сборка занимает ~5–8 минут).
5. Откройте завершённый запуск → внизу страницы раздел **Artifacts** →
   скачайте `tpager-gps-wifi-tracker-bin.zip`. Внутри — готовый `.bin`.
6. Прошейте плату через любой инструмент прошивки ESP32 (например,
   [ESP Web Tools](https://esphome.github.io/esp-web-tools/) прямо в браузере,
   без установки чего-либо — просто откройте плату через USB и загрузите .bin),
   либо через `esptool.py write_flash 0x0 firmware.bin`, если Python уже стоит.

## Если сборка упадёт на шаге "Compile firmware"

Самая вероятная причина — я не смог на 100% гарантированно узнать точное имя FQBN
(идентификатор платы) `LilyGo-T-LoRa-Pager` в файле `boards.txt` пакета arduino-esp32 —
LILYGO могли назвать её чуть иначе внутри самого пакета. Workflow уже содержит шаг
**"List available T-LoRa-Pager board FQBNs"**, который выведет точный список — откройте
его лог, скопируйте настоящий FQBN оттуда и одной строкой поправьте
`.github/workflows/build.yml` (строка с `--fqbn "esp32:esp32:...`), закоммитьте — сборка
перезапустится сама.

## Прошивка через браузер (без esptool/Python)

Откройте https://esphome.github.io/esp-web-tools/ в Chrome/Edge, подключите T-LoRa Pager
по USB-C, выберите скачанный `.bin`, укажите адрес `0x0` — прошивка зальётся прямо из
браузера, без установки ПО.
