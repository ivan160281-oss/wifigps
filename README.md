# GPS + WiFi трекер для LILYGO T-LoRa Pager

## Как это работает

**Стартовый экран (запись не идёт):**
- статус GPS (поиск / фикс получен, кол-во спутников)
- статус WiFi (готов, сколько сетей видно прямо сейчас)
- текущее время UTC+3 (берётся с GPS)
- подсказка `ENTER - начать запись` (используется встроенная клавиатура T-LoRa Pager)

**ENTER** на стартовом экране → начинается запись.
**ENTER** во время записи → запись останавливается (файл закрывается через flush), возврат на стартовый экран.
**E** (в любой момент, кроме уже остановленного состояния) → полная остановка приложения: файл закрывается, на экране надпись «Приложение остановлено», дальше устройство ничего не делает (нужна перезагрузка).

**Во время записи на экране:**
- красный трек на чёрном фоне (нижняя часть экрана)
- таблица: кол-во спутников, координаты, кол-во видимых WiFi сетей, скорость (среднее по последним 5 точкам, км/ч), размер текущего файла в КБ
- счётчики: сколько строк записано за сессию, сколько сетей просканировано суммарно за сессию

**На SD-карте:**
- папка `/WIFIGPS` в корне
- новый файл каждые 30 минут записи: `log_ГГГГММДД_ЧЧММСС.txt` (имя — по времени UTC+3 на момент создания файла)
- строка каждые 30 секунд, формат: `HH:MM:SS_lat_lon_wifi1,wifi2,wifi3` (если сетей не видно — пусто после последнего `_`)
- без валидных координат (GPS без фикса) строка не пишется, но 30-секундный таймер и 30-минутная ротация файлов продолжают идти

**WiFi-сканирование** — асинхронное, раз в 15 сек, не блокирует ни GPS, ни экран.

Файл: `gps_wifi_tracker/gps_wifi_tracker.ino`

## Важно про то, что нет .bin здесь

Скомпилировать `.bin` без установки чего-либо локально можно через GitHub Actions —
уже подготовленный `.github/workflows/build.yml` делает это автоматически при пуше
в ветку `main`/`master`. Собирается на стабильном ESP32-ядре 3.3.10.

### Как получить .bin
1. Создайте репозиторий на GitHub, загрузите файлы отсюда, сохранив пути:
   ```
   .github/workflows/build.yml
   gps_wifi_tracker/gps_wifi_tracker.ino
   ```
2. После коммита откройте вкладку **Actions** — сборка запустится сама (~5-8 мин)
3. Скачайте `tpager-gps-wifi-tracker-bin.zip` из **Artifacts** завершённого запуска
4. Внутри — 5 файлов; проще всего прошить **`gps_wifi_tracker.ino.merged.bin`** на адрес `0x0`
   (например, через https://espressif.github.io/esptool-js/ в браузере, без установки ПО)

## Технические детали (для разработчика)

- Библиотека: [LilyGoLib](https://github.com/Xinyuan-LilyGO/LilyGoLib) (официальная для T-LoRa Pager)
- Плата (fqbn): `esp32:esp32:tlora_pager`
- GPS уже встроен в библиотеку как `instance.gps` (наследует TinyGPSPlus), UART настраивается автоматически внутри `instance.begin()` на 38400 бод
- Клавиатура: `instance.kb.getKey(&c)` — физическая QWERTY-клавиатура платы
- SD монтируется библиотекой в точку `/sd` через `instance.installSD()`
- Экран рисуется через LVGL (v9), трек — через `lv_canvas` с прямым рисованием линий
- Минимум спутников для признания фикса валидным — 3 (`MIN_SATS_FOR_FIX`, можно ужесточить)
